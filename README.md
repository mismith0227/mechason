# mechason

[ぞうAdvent Calendar](http://www.adventar.org/calendars/759"ぞうアドベントカレンダー")18日目です。

enchant.jsで作りました。PCでご覧ください。
ホントはもうちょっと、ゲーム要素も加えたかったけど。。。

## 操作方法

PCのキーボードで操作してください。

- ← / → : 左右に移動
- ↑ : 上昇
- z : 酒を投げる

左右からぞうが飛んできます。
当たると飛ばされます。

[Github Page](http://mismith0227.github.io/mechason/)でお楽しみください。
